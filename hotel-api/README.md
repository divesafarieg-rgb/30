# Hotel API - Реализация веб-сервиса

## Описание
Реализовано API для системы бронирования отелей по контракту заказчика.
Использована TDD-разработка через тестирование.

## Реализованные эндпоинты:
1. `GET /room` - получение списка всех комнат
2. `POST /add-room` - добавление новой комнаты
3. `POST /booking` - бронирование комнаты

## Технологии:
- Python 3.x
- Flask (веб-фреймворк)
- SQLite (база данных)
- SQLAlchemy (опционально, для работы с БД)

## Тестирование:
Все тесты из Postman коллекции успешно пройдены:
- GET /room: проверка статуса 200 и структуры ответа
- POST /add-room: проверка добавления комнаты
- POST /booking: проверка бронирования и обработки конфликтов

## Особенности реализации:
- Модели данных вынесены в отдельный модуль
- Бизнес-логика отделена от API слоя
- Используется база данных для хранения состояния
- Корректная обработка HTTP статусов (200, 409)
